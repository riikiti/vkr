# 1. Цель практической части

Целью практической части является разработка программной системы для
экспериментального анализа криптостойкости алгоритмов шифрования на
основе методов энтропийного анализа.

Система должна позволять:

-   шифровать данные различными алгоритмами
-   вычислять энтропийные характеристики
-   анализировать статистические свойства шифртекста
-   проводить тест лавинного эффекта
-   визуализировать результаты исследований
-   формировать аналитические отчёты

# 3. Используемые методы анализа

## 3.1 Энтропия Шеннона

Формула:

H = - Σ p(x) log₂ p(x)

где p(x) --- вероятность появления символа.

Максимальная энтропия для байтовых данных: **8 бит**.

------------------------------------------------------------------------

## 3.2 Условная энтропия

Используется для оценки зависимости между исходным текстом и
шифртекстом.

------------------------------------------------------------------------

## 3.3 Дивергенция Кульбака--Лейблера

Позволяет оценить различие между распределениями вероятностей.

------------------------------------------------------------------------

## 3.4 Взаимная информация

Определяет степень статистической зависимости между входными и выходными
данными.

------------------------------------------------------------------------

## 3.5 Тест лавинного эффекта

Изменение одного бита входного сообщения должно приводить к изменению
примерно 50% бит шифртекста.

Метрика:

Avalanche = changed_bits / total_bits

# 18. Критерии оценки криптостойкости

Алгоритм считается более криптостойким если:

-   энтропия близка к 8
-   распределение байтов равномерное
-   корреляция минимальная
-   лавинный эффект ≈ 0.5
