# 01. Цели и задачи практической части

**Тема ВКР:** Исследование криптостойкости алгоритмов шифрования с применением энтропийного анализа
**Тип работы:** Магистерская ВКР
**Направление:** Информатика и вычислительная техника

---

## 1.1 Цель

Разработка программной системы для экспериментального анализа криптостойкости алгоритмов шифрования на основе методов энтропийного анализа.

## 1.3 Исследуемые алгоритмы шифрования

| Алгоритм | Тип | Длина ключа | Режим |
|----------|-----|-------------|-------|
| AES | Блочный | 128/192/256 бит | CBC / GCM |
| DES | Блочный | 56 бит | CBC |
| Blowfish | Блочный | 32–448 бит | CBC |
| RC4 | Потоковый | 40–2048 бит | — |
| ChaCha20 | Потоковый | 256 бит | — |

**Примечание к реализации:** DES и RC4 считаются устаревшими с точки зрения безопасности, однако включены в исследование в качестве контрольных алгоритмов с заведомо низкой криптостойкостью (особенно DES с 56-битным ключом), что позволяет валидировать метрики.

---

## 1.4 Методы анализа криптостойкости

### Энтропийные метрики
- **Энтропия Шеннона** — основная метрика случайности шифртекста
- **Условная энтропия** — оценка зависимости открытого текста и шифртекста
- **Дивергенция Кульбака–Лейблера** — отклонение распределения от равномерного
- **Взаимная информация** — степень статистической зависимости входа и выхода

### Статистические метрики
- Анализ частот байтов (frequency analysis)
- Корреляционный анализ
- Дисперсия, асимметрия, эксцесс распределения байтов

### Функциональные тесты
- **Тест лавинного эффекта** — изменение одного бита на входе → ~50 % изменённых бит на выходе

---

## 1.5 Критерии оценки криптостойкости

| Метрика | Идеальное значение | Интерпретация |
|---------|-------------------|---------------|
| Энтропия Шеннона | ≈ 8.0 бит/байт | Максимальная случайность |
| KL-дивергенция | ≈ 0 | Распределение близко к равномерному |
| Взаимная информация | ≈ 0 | Отсутствие зависимости открытый↔шифртекст |
| Лавинный коэффициент | ≈ 0.5 (50 %) | Соответствие критерию лавинного эффекта |
| Корреляция байтов | ≈ 0 | Отсутствие линейной зависимости |

## 1.6 Ожидаемые результаты

- Сравнительные таблицы метрик по каждому алгоритму
- Графики энтропии в зависимости от алгоритма и размера данных
- Гистограммы распределения байтов шифртекста
- Графики лавинного эффекта
- Аналитический отчёт в форматах CSV / JSON / Markdown
- Интерактивный веб-дашборд (Streamlit)

---

## 1.7 Научная ценность

Практическая часть позволяет:
1. Экспериментально подтвердить теоретические свойства алгоритмов
2. Сравнить современные и устаревшие алгоритмы по единой методике
3. Показать применимость энтропийного анализа как инструмента оценки криптостойкости
4. Сформировать воспроизводимую исследовательскую систему
