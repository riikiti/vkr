## 8.1 Теория лавинного эффекта

**Лавинный эффект (Avalanche Effect)** — фундаментальное свойство криптографических алгоритмов: изменение одного бита входных данных должно приводить к изменению примерно **50 % бит** шифртекста.

### Формула

```
Avalanche(i) = changed_bits(i) / total_bits
```

где:
- `changed_bits(i)` — количество изменённых бит при инверсии i-го бита входа
- `total_bits` — общее число бит в шифртексте

**Идеальный результат:** `Avalanche ≈ 0.5` (50 %)

**Интерпретация:**
- `< 0.4` — слабый лавинный эффект, алгоритм уязвим
- `0.4–0.6` — хороший лавинный эффект
- `≈ 0.5` — идеальный лавинный эффект (AES, ChaCha20)

---

## 8.2 Алгоритм теста

```
Вход: алгоритм A, данные D, ключ K, число итераций N

Для i = 1..N:
  1. Выбрать случайный бит-позицию pos в D
  2. Создать D' = D с инвертированным битом pos
  3. C  = A.encrypt(D, K)
  4. C' = A.encrypt(D', K)
  5. Вычислить hamming_distance(C, C') / (len(C) * 8)
  6. Сохранить коэффициент

Результат: среднее, дисперсия, гистограмма коэффициентов
```

## 8.3 Реализация `avalanche_test.py`

```python
# core/avalanche/avalanche_test.py
import numpy as np
from config import AVALANCHE_TESTS, AVALANCHE_IDEAL


def flip_bit(data: bytes, bit_pos: int) -> bytes:
    """
    Инвертирует один бит в байтовой строке.
    bit_pos: позиция бита (0 = старший бит первого байта).
    """
    byte_idx = bit_pos // 8
    bit_idx = 7 - (bit_pos % 8)   # MSB first
    data_list = bytearray(data)
    data_list[byte_idx] ^= (1 << bit_idx)
    return bytes(data_list)


def hamming_distance_bytes(a: bytes, b: bytes) -> int:
    """
    Расстояние Хэмминга между двумя байтовыми строками (число различающихся бит).
    """
    min_len = min(len(a), len(b))
    a_arr = np.frombuffer(a[:min_len], dtype=np.uint8)
    b_arr = np.frombuffer(b[:min_len], dtype=np.uint8)
    xor = np.bitwise_xor(a_arr, b_arr)
    # Подсчёт установленных бит через popcount
    return int(np.unpackbits(xor).sum())


def avalanche_coefficient(cipher_1: bytes, cipher_2: bytes) -> float:
    """
    Коэффициент лавинного эффекта для одной пары шифртекстов.
    """
    total_bits = min(len(cipher_1), len(cipher_2)) * 8
    if total_bits == 0:
        return 0.0
    changed = hamming_distance_bytes(cipher_1, cipher_2)
    return changed / total_bits


def run_avalanche_test(
    cipher,            # BaseCipher instance
    data: bytes,
    key: bytes,
    n_tests: int = AVALANCHE_TESTS,
    random_seed: int = 42
) -> dict:
    """
    Запускает полный тест лавинного эффекта для одного алгоритма.

    Аргументы:
        cipher:    экземпляр шифра (BaseCipher)
        data:      исходные данные
        key:       ключ шифрования
        n_tests:   число итераций (по умолчанию из config.py)
        random_seed: для воспроизводимости

    Возвращает dict с:
        mean:       средний коэффициент
        std:        стандартное отклонение
        min_val:    минимальный коэффициент
        max_val:    максимальный коэффициент
        coefficients: список всех коэффициентов
        is_good:    True если mean ≈ 0.5 (отклонение < 0.1)
    """
    rng = np.random.default_rng(seed=random_seed)
    total_bits = len(data) * 8

    if total_bits == 0:
        raise ValueError("Данные не могут быть пустыми")

    # Базовый шифртекст
    ciphertext_base = cipher.encrypt(data, key)
    coefficients = []

    for _ in range(n_tests):
        # Случайная позиция бита
        bit_pos = int(rng.integers(0, total_bits))
        # Данные с инвертированным битом
        data_flipped = flip_bit(data, bit_pos)
        # Шифрование изменённых данных
        ciphertext_flipped = cipher.encrypt(data_flipped, key)
        # Коэффициент лавинного эффекта
        coef = avalanche_coefficient(ciphertext_base, ciphertext_flipped)
        coefficients.append(coef)

    coefficients = np.array(coefficients)

    return {
        "algorithm": cipher.name,
        "n_tests": n_tests,
        "mean": float(np.mean(coefficients)),
        "std": float(np.std(coefficients)),
        "min_val": float(np.min(coefficients)),
        "max_val": float(np.max(coefficients)),
        "coefficients": coefficients.tolist(),
        "is_good": bool(abs(np.mean(coefficients) - AVALANCHE_IDEAL) < 0.1),
    }


def run_avalanche_test_per_bit(
    cipher,
    data: bytes,
    key: bytes,
    sample_bits: int = 64
) -> dict:
    """
    Тест лавинного эффекта для каждого из sample_bits позиций.
    Позволяет построить тепловую карту: какие биты влияют сильнее.

    Возвращает:
        bit_positions: список позиций бит
        coefficients:  соответствующие коэффициенты
    """
    ciphertext_base = cipher.encrypt(data, key)
    total_bits = min(len(data) * 8, sample_bits)
    positions = list(range(total_bits))
    coefficients = []

    for pos in positions:
        data_flipped = flip_bit(data, pos)
        ciphertext_flipped = cipher.encrypt(data_flipped, key)
        coef = avalanche_coefficient(ciphertext_base, ciphertext_flipped)
        coefficients.append(coef)

    return {
        "bit_positions": positions,
        "coefficients": coefficients,
    }
```

| Алгоритм | Ожидаемый Avalanche | Примечание |

| RC4 | 0.40–0.48 | Слабее из-за потокового характера |

## 8.5 Тест на разных размерах данных

Важно: результаты лавинного эффекта зависят от размера данных. Блочные шифры в режиме CBC — изменение одного бита влияет только на текущий и все последующие блоки.

```python
def avalanche_vs_data_size(cipher, sizes: list[int]) -> list[dict]:
    """
    Запускает тест для разных размеров входных данных.
    Позволяет исследовать зависимость лавинного эффекта от размера.
    """
    from data_generation.random_generator import generate_random_data
    results = []
    for size in sizes:
        data = generate_random_data(size)
        key = cipher.generate_key()
        result = run_avalanche_test(cipher, data, key)
        result["data_size"] = size
        results.append(result)
    return results
```
