# 04. Модуль генерации данных

**Папка:** `data_generation/`

---

## 4.1 Назначение модуля

Модуль создаёт входные данные различных типов для тестирования алгоритмов шифрования. Разнообразие типов данных важно потому, что разные алгоритмы могут вести себя по-разному на данных с различной начальной энтропией.

**Ключевой принцип:** все генераторы возвращают `bytes` заданного размера, а сам тип данных влияет на начальную энтропию — это исследовательский параметр.

---

## 4.2 Общий интерфейс

```python
# data_generation/__init__.py

def generate_data(data_type: str, size: int, **kwargs) -> bytes:
    """
    Фабричная функция. data_type: 'text' | 'binary' | 'random' | 'image'
    """
    generators = {
        "text": generate_text_data,
        "binary": generate_binary_data,
        "random": generate_random_data,
        "image": generate_image_data,
    }
    return generators[data_type](size, **kwargs)
```

## 4.3 `text_generator.py` — Генерация текстовых данных

**Цель:** данные с характерным распределением букв естественного языка (низкая энтропия ~4–5 бит/байт), что позволяет наблюдать, как шифрование приближает распределение к равномерному.

```python
# data_generation/text_generator.py
import random
import string

def generate_plain_text(size: int) -> bytes:
    """ASCII-символы: буквы + цифры + пробелы."""
    chars = string.ascii_letters + string.digits + " " * 10  # пробел часто
    text = "".join(random.choices(chars, k=size))
    return text.encode("ascii")[:size]

def generate_natural_language_text(size: int) -> bytes:
    """
    Имитация естественного языка через повтор слов.
    Высокая повторяемость → низкая энтропия исходника.
    """
    words = [
        "the", "and", "is", "in", "it", "of", "to", "a",
        "that", "this", "with", "for", "on", "are", "be",
    ]
    result = []
    while len(" ".join(result).encode()) < size:
        result.append(random.choice(words))
    return " ".join(result).encode()[:size]

def generate_structured_text(size: int) -> bytes:
    """Структурированные данные (JSON-подобный формат)."""
    template = '{"id": %d, "value": "%s", "flag": %s}\n'
    result = b""
    i = 0
    while len(result) < size:
        row = (template % (i, "data", "true")).encode()
        result += row
        i += 1
    return result[:size]
```

**Ожидаемая энтропия исходного текста:** 4.0–5.5 бит/байт

## 4.4 `binary_generator.py` — Бинарные последовательности

**Цель:** данные с контролируемой энтропией — паттерны и повторяющиеся структуры.

```python
# data_generation/binary_generator.py
import numpy as np

def generate_binary_sequence(size: int, pattern_len: int = 8) -> bytes:
    """
    Повторяющийся бинарный паттерн. Низкая энтропия.
    pattern_len: длина базового паттерна в байтах.
    """
    pattern = np.random.randint(0, 256, size=pattern_len, dtype=np.uint8)
    repeats = (size // pattern_len) + 1
    data = np.tile(pattern, repeats)[:size]
    return bytes(data)

def generate_zero_data(size: int) -> bytes:
    """Нулевые данные. Минимальная энтропия = 0."""
    return b"\x00" * size

def generate_incremental_data(size: int) -> bytes:
    """Счётчик 0..255 по кругу. Равномерное, но не случайное."""
    data = bytes([i % 256 for i in range(size)])
    return data
```

**Ожидаемая энтропия:** 0–3 бит/байт (зависит от паттерна)

## 4.5 `image_generator.py` — Данные изображений

**Цель:** симуляция работы с мультимедиа-данными. Пиксельные данные имеют собственное распределение яркостей.

```python
# data_generation/image_generator.py
import numpy as np

def generate_random_image(size: int) -> bytes:
    """
    Пиксельные данные случайного изображения (RGB).
    Высокая энтропия исходника ~7–8 бит/байт.
    """
    pixels = np.random.randint(0, 256, size=size, dtype=np.uint8)
    return bytes(pixels)

def generate_pattern_image(size: int) -> bytes:
    """
    Изображение с повторяющимися паттернами (горизонтальные полосы).
    Средняя энтропия ~5–6 бит/байт.
    """
    row_size = 64  # ширина строки в байтах
    num_colors = 4
    colors = np.random.randint(0, 256, size=num_colors, dtype=np.uint8)
    result = b""
    i = 0
    while len(result) < size:
        color = colors[i % num_colors]
        result += bytes([color] * row_size)
        i += 1
    return result[:size]
```

## 4.6 `random_generator.py` — Случайные данные (контрольное распределение)

**Цель:** контрольный образец с заведомо максимальной энтропией (~8 бит/байт). Используется для сравнения: хорошее шифрование должно давать результат, близкий к случайным данным.

```python
# data_generation/random_generator.py
import os
import numpy as np

def generate_random_data(size: int, use_os_random: bool = True) -> bytes:
    """
    Криптографически случайные данные.
    use_os_random=True: использует os.urandom() (CSPRNG)
    use_os_random=False: numpy random (для воспроизводимости)
    """
    if use_os_random:
        return os.urandom(size)
    else:
        rng = np.random.default_rng(seed=42)
        return bytes(rng.integers(0, 256, size=size, dtype=np.uint8))
```

**Ожидаемая энтропия:** ≈ 8.0 бит/байт

## 4.7 Таблица сравнения типов данных

| Тип | Генератор | Ожидаемая энтропия | Назначение в эксперименте |
|-----|-----------|-------------------|--------------------------|
| Текст (ASCII) | `generate_plain_text` | 4–5 бит/байт | Базовый тест |
| Текст (natural) | `generate_natural_language_text` | 3–4 бит/байт | Тест на структурированном входе |
| Бинарный паттерн | `generate_binary_sequence` | 0–3 бит/байт | Тест на минимальной энтропии |
| Изображение (random) | `generate_random_image` | 7–8 бит/байт | Тест на высокой исходной энтропии |
| Изображение (pattern) | `generate_pattern_image` | 5–6 бит/байт | Тест на структурированном медиа |
| Случайные данные | `generate_random_data` | ≈ 8 бит/байт | Контрольный образец |

## 4.8 Сохранение сгенерированных данных

```python
import os
from config import GENERATED_DIR

def save_generated_data(data: bytes, data_type: str, size: int) -> str:
    filename = f"{data_type}_{size}.bin"
    path = os.path.join(GENERATED_DIR, filename)
    with open(path, "wb") as f:
        f.write(data)
    return path
```
