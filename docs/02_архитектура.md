## 2.2 Слои архитектуры

Система разделена на **5 архитектурных слоёв**:

| Слой | Модули | Ответственность |
|------|--------|-----------------|
| **UI** | `interface/` | Пользовательский интерфейс, параметры запуска |
| **Experiments** | `experiments/` | Оркестрация, управление экспериментами |
| **Core** | `core/encryption/`, `core/entropy/`, `core/statistics/`, `core/avalanche/` | Вычисления |
| **Analytics** | `analytics/` | Агрегация метрик, генерация отчётов |
| **Visualization** | `visualization/` | Построение графиков и дашбордов |

## 2.4 Паттерны проектирования

### Strategy (Стратегия) — для алгоритмов шифрования
Все шифры реализуют единый интерфейс `BaseCipher`, что позволяет их взаимозаменяемо использовать в экспериментах.

```python
# core/encryption/base_cipher.py
from abc import ABC, abstractmethod

class BaseCipher(ABC):
    @abstractmethod
    def encrypt(self, data: bytes, key: bytes) -> bytes: ...

    @abstractmethod
    def decrypt(self, data: bytes, key: bytes) -> bytes: ...

    @abstractmethod
    def generate_key(self) -> bytes: ...

    @property
    @abstractmethod
    def name(self) -> str: ...
```

### Factory (Фабрика) — для создания шифров по имени
```python
# core/encryption/__init__.py
def get_cipher(name: str) -> BaseCipher:
    registry = {
        "AES": AESCipher,
        "DES": DESCipher,
        "BLOWFISH": BlowfishCipher,
        "RC4": RC4Cipher,
        "CHACHA20": ChaCha20Cipher,
    }
    return registry[name.upper()]()
```

### Template Method — для экспериментов
Базовый класс `BaseExperiment` задаёт шаблон выполнения, конкретные эксперименты переопределяют шаги.

## 2.6 Форматы данных между модулями

| Передача | Тип данных | Описание |
|----------|-----------|----------|
| Генератор → Шифрование | `bytes` | Исходные данные |
| Шифрование → Анализ | `bytes` | Шифртекст |
| Анализ → Эксперименты | `dict[str, float]` | Метрики |
| Эксперименты → Аналитика | `pd.DataFrame` | Сводные результаты |
| Аналитика → Визуализация | `pd.DataFrame` | Данные для графиков |

## 2.7 Точки расширения

- **Добавление нового алгоритма:** создать класс-наследник `BaseCipher`, зарегистрировать в фабрике
- **Добавление новой метрики:** создать функцию в `core/entropy/` или `core/statistics/`, включить в `metrics_calculator.py`
- **Добавление нового эксперимента:** создать наследник `BaseExperiment`, добавить в `experiment_runner.py`
