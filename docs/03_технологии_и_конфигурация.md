## 3.3 Файл конфигурации `config.py`

```python
# config.py

# Размеры тестовых данных (в байтах)
DATA_SIZES = [
    1024,       # 1 KB
    10240,      # 10 KB
    102400,     # 100 KB
    1048576,    # 1 MB

# Список исследуемых алгоритмов
ALGORITHMS = [
    "AES",

    "BLOWFISH",
    "RC4",
    "CHACHA20",

    "text",     # Текстовые данные
    "binary",   # Бинарные последовательности
```

## 3.5 Обоснование выбора библиотек

### pycryptodome vs cryptography
`pycryptodome` выбран потому что:
- Предоставляет низкоуровневый доступ к алгоритмам (DES, Blowfish, RC4) — в библиотеке `cryptography` часть из них deprecated
- Единый API для всех алгоритмов (включая устаревшие)
- Подходит для исследовательских целей без production-ограничений

### plotly vs только matplotlib
`plotly` используется для интерактивных графиков в Streamlit-интерфейсе. `matplotlib`/`seaborn` — для статических изображений в отчётах.

### pandas для результатов
Все результаты экспериментов хранятся как `pd.DataFrame`, что упрощает агрегацию, сортировку, экспорт в CSV/JSON и передачу в Streamlit-таблицы.

## 3.6 Версионирование и воспроизводимость

- Фиксировать версии в `requirements.txt` (через `pip freeze > requirements.txt` после настройки)
- Все случайные числа генерируются с `random_seed` из `config.py` для воспроизводимости
- Результаты экспериментов сохраняются в `data/results/` с временными метками
