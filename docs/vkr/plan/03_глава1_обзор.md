# Глава 1. Анализ подходов к оценке криптостойкости алгоритмов шифрования

**Статус:** ⬜ Первая глава к написанию

## Структура главы

### 1.1. Основные понятия криптографии и криптоанализа (3-4 стр.)

**Подзадачи:**
- [ ] Определение криптографии, шифрования, дешифрования
- [ ] Классификация алгоритмов шифрования:
  - Симметричные vs асимметричные
  - Блочные vs потоковые
- [ ] Понятие криптостойкости
  - Определение (стойкость к криптоанализу)
  - Виды криптоанализа: силовой перебор, дифференциальный, линейный, алгебраический
  - Понятие вычислительной стойкости
- [ ] Стандарты и требования к криптоалгоритмам (NIST, ГОСТ)

**Источники:** учебники по криптографии (Шнайер, Менезес, ГОСТ Р 34.12-2018)

### 1.2. Обзор исследуемых алгоритмов шифрования (5-6 стр.)

**Подзадачи:**
- [ ] AES (Advanced Encryption Standard)
  - История создания (конкурс NIST, Rijndael)
  - Структура: SPN (Substitution-Permutation Network)
  - Размеры ключа (128/192/256 бит), размер блока (128 бит)
  - Области применения
- [ ] DES (Data Encryption Standard)
  - История, сеть Фейстеля
  - Уязвимости (малый размер ключа 56 бит)
  - Причины вывода из эксплуатации
- [ ] 3DES (Triple DES)
  - Схема EDE (Encrypt-Decrypt-Encrypt)
  - Улучшения относительно DES
- [ ] Blowfish
  - Структура (сеть Фейстеля, S-блоки)
  - Переменный размер ключа
- [ ] RC4
  - Потоковый шифр, принцип работы
  - Известные уязвимости (bias в начальных байтах)
- [ ] ChaCha20
  - Современный потоковый шифр (Бернстайн)
  - ARX-операции (Add, Rotate, XOR)
  - Преимущества перед RC4
- [ ] ГОСТ 28147-89
  - Российский стандарт шифрования
  - Сеть Фейстеля, 32 раунда, S-блоки
  - Соотношение с ГОСТ Р 34.12-2018 (Магма)

**Источники:** RFC, стандарты NIST/ГОСТ, оригинальные публикации

### 1.3. Существующие подходы к оценке криптостойкости (3-4 стр.)

**Подзадачи:**
- [ ] Формальные методы (доказуемая безопасность)
  - Модели безопасности (IND-CPA, IND-CCA)
  - Ограничения формальных доказательств
- [ ] Статистические тесты (NIST SP 800-22)
  - Тесты NIST для оценки случайности
  - Тесты Diehard, TestU01
- [ ] Энтропийный анализ
  - Связь энтропии и криптостойкости
  - Существующие работы по применению энтропии в криптоанализе
- [ ] Тест лавинного эффекта (SAC — Strict Avalanche Criterion)
  - Определение и формула
  - Связь с криптостойкостью
- [ ] Анализ программных решений (аналогов)
  - CrypTool
  - SageMath
  - Другие инструменты
  - Сравнительная таблица

**Источники:** NIST SP 800-22, статьи по криптоанализу, Shannon 1949

### 1.4. Постановка задачи исследования (2-3 стр.)

**Подзадачи:**
- [ ] Выявление ограничений существующих подходов
  - Формальные методы: сложность, не всегда практичны
  - NIST-тесты: бинарный результат (pass/fail), нет ранжирования
  - Отсутствие комплексного инструмента с энтропийными метриками
- [ ] Обоснование применения энтропийного анализа
  - Теоретическая связь энтропии и криптостойкости (Шеннон, 1949)
  - Количественная оценка (не бинарная)
  - Возможность сравнения алгоритмов
- [ ] Формулирование требований к программному решению
  - Поддержка множества алгоритмов
  - Множество типов входных данных
  - Визуализация результатов
  - Генерация отчетов
