## Структура заключения (по требованиям, стр. 29 PDF)

### Основные выводы (5-6 пунктов, нумерованные)

1. Проведен анализ существующих подходов к оценке криптостойкости алгоритмов шифрования. Выявлены ограничения формальных методов и стандартных статистических тестов, обоснована необходимость комплексного подхода на основе энтропийного анализа.

2. Систематизированы и исследованы энтропийные метрики (энтропия Шеннона, KL-дивергенция, условная энтропия, взаимная информация), установлены пороговые значения для оценки качества шифрования.

3. Разработана комплексная методика оценки криптостойкости, объединяющая энтропийные метрики со статистическими тестами и тестом лавинного эффекта, с агрегированным показателем для ранжирования алгоритмов.

4. Спроектирована и реализована веб-платформа CryptoAnalyzer для автоматизированного анализа криптостойкости с поддержкой 7 алгоритмов шифрования, 7 типов входных данных и визуализацией результатов.

5. Проведена экспериментальная проверка методики, подтвердившая ее работоспособность. Получены количественные оценки криптостойкости для AES, DES, 3DES, Blowfish, RC4, ChaCha20 и ГОСТ 28147-89, сформулированы рекомендации по выбору алгоритмов.
