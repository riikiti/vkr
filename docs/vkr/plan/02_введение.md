# Введение

**Статус:** ⬜ Пишется после основных глав (кроме краткого содержания глав)

## Структура введения (по требованиям ИВТ, стр. 9 PDF)

### 2.1. Краткая характеристика предметной области (0.5 стр.)
- Роль криптографии в современном мире
- Цифровая трансформация, рост объемов данных, киберугрозы
- Необходимость объективной оценки криптостойкости алгоритмов шифрования

### 2.2. Актуальность темы (0.5 стр.)
- Рост количества кибератак и утечек данных
- Появление новых угроз (квантовые компьютеры, side-channel attacks)
- Необходимость количественных методов оценки криптостойкости помимо формальных доказательств
- Энтропийный анализ как инструмент, дополняющий классические методы криптоанализа
- Отсутствие комплексных инструментов для сравнительного анализа алгоритмов

### 2.3. Цель и задачи работы (0.5 стр.)

**Цель:** исследование и сравнительная оценка криптостойкости алгоритмов симметричного шифрования с применением комплексного энтропийного анализа.

**Задачи:**
1. Анализ существующих подходов к оценке криптостойкости алгоритмов шифрования и выявление их ограничений
2. Исследование и систематизация энтропийных метрик, применимых для оценки криптостойкости (энтропия Шеннона, расхождение Кульбака-Лейблера, условная энтропия, взаимная информация)
3. Разработка комплексной методики оценки криптостойкости, объединяющей энтропийные метрики со статистическими тестами и тестом лавинного эффекта
4. Проектирование и разработка программной платформы CryptoAnalyzer для автоматизированного анализа криптостойкости
5. Экспериментальная проверка разработанной методики на 7 алгоритмах шифрования (AES, DES, 3DES, Blowfish, RC4, ChaCha20, ГОСТ 28147-89) с различными типами входных данных
6. Сравнительный анализ результатов и формулирование рекомендаций по выбору алгоритмов шифрования

### 2.4. Объект и предмет исследования
- **Объект:** алгоритмы симметричного шифрования (блочные и потоковые)
- **Предмет:** методы оценки криптостойкости алгоритмов шифрования на основе энтропийного анализа

### 2.5. Методы исследования
- Теория информации (энтропия Шеннона, теория кодирования)
- Математическая статистика (критерий хи-квадрат, корреляционный анализ)
- Теория вероятностей (распределения случайных величин)
- Криптографический анализ (лавинный эффект, частотный анализ)
- Методы программной инженерии (объектно-ориентированное проектирование)

### 2.6. Теоретическая значимость
- Разработка комплексной методики оценки криптостойкости, объединяющей 4 энтропийные метрики с набором статистических тестов и тестом лавинного эффекта
- Систематизация критериев качества шифрования на основе информационно-теоретических показателей

### 2.7. Практическая значимость
- Программная платформа CryptoAnalyzer для автоматизированного анализа криптостойкости
- Результаты сравнительного анализа 7 алгоритмов как основа для рекомендаций по выбору алгоритмов

### 2.8. Краткое описание структуры работы (0.5 стр.)
- «В первой главе…» — обзор предметной области, постановка задачи
- «Во второй главе…» — теоретические основы, методика оценки
- «В третьей главе…» — программная реализация платформы
- «В четвертой главе…» — экспериментальные результаты и выводы
