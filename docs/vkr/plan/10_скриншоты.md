## Как получить скриншоты

### Вариант 1: Через WebFetch (автоматически)
- Обращение к http://91.229.8.171/ — но SPA не рендерится в HTML

### Вариант 2: Запросить у пользователя
- Попросить пользователя сделать скриншоты вручную
- Указать точные параметры эксперимента для воспроизведения

### Вариант 3: Получить данные через API и описать текстово
- POST http://91.229.8.171/api/experiments/run — получить числовые данные
- Описать результаты в таблицах
- Скриншоты попросить у пользователя

## Рекомендуемые параметры для скриншотов
- **Алгоритмы:** все 7 (AES, DES, 3DES, Blowfish, RC4, ChaCha20, ГОСТ)
- **Типы данных:** все 7
- **Размеры:** 1024, 10240, 102400
- **Итерации лавинного теста:** 100
- Это даст полную картину для всех скриншотов
