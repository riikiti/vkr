# 4.1. Методика проведения экспериментов

> Статус: ✅ Готово

## Экспериментальная установка

Экспериментальные исследования проводились на платформе CryptoAnalyzer, развернутой на виртуальном сервере (VPS) с операционной системой Linux. Система доступна по адресу http://91.229.8.171/. Развертывание выполнено с использованием Docker Compose, что обеспечивает воспроизводимость окружения.

## Параметры экспериментов

В экспериментах исследовались шесть алгоритмов шифрования: AES-256, DES, 3DES, Blowfish, RC4 и ГОСТ 28147-89. Для каждого алгоритма использовались семь типов тестовых данных: нулевые байты (zeros), бинарный паттерн (binary), структурированные данные (structured), естественный текст (text), изображение (image), инкрементальная последовательность (incremental) и криптографически случайные данные (random). Данные генерировались в трех размерах: 1 КБ, 10 КБ и 100 КБ. Количество итераций теста лавинного эффекта — 100.

## План экспериментов

Общее количество комбинаций для энтропийного и статистического анализа: 6 алгоритмов × 7 типов данных × 3 размера = 126 экспериментов. Для теста лавинного эффекта: 6 алгоритмов × 3 размера = 18 серий по 100 итераций каждая. Итого проведено 1944 итерации лавинного теста.

Все эксперименты выполнены через REST API платформы (POST /api/experiments/run) с фиксированными параметрами, что обеспечивает воспроизводимость результатов.
