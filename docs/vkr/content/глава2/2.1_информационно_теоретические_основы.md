# 2.1. Информационно-теоретические основы оценки криптостойкости

> Статус: ✅ Готово

Связь между теорией информации и криптографией была установлена К. Шенноном в фундаментальной работе «Communication Theory of Secrecy Systems» (1949). Шеннон показал, что задача шифрования может быть формализована в терминах теории информации, а криптостойкость алгоритма может быть оценена через информационно-теоретические характеристики шифртекста.

Центральным понятием теории информации является **энтропия** — мера неопределенности (количества информации) случайной величины. Шеннон определил энтропию дискретной случайной величины X, принимающей значения x₁, x₂, ..., xₙ с вероятностями p(x₁), p(x₂), ..., p(xₙ), формулой:

**H(X) = −∑ᵢ p(xᵢ) · log₂(p(xᵢ))**                  (2.1)

Энтропия обладает следующими свойствами:
- неотрицательность: H(X) ≥ 0;
- H(X) = 0 тогда и только тогда, когда одно из значений имеет вероятность 1 (полная определенность);
- максимум достигается при равномерном распределении: H(X) ≤ log₂(n), причем равенство выполняется при p(xᵢ) = 1/n для всех i.

Для байтовой последовательности (n = 256 возможных значений) максимальная энтропия составляет:

**H_max = log₂(256) = 8 бит/байт**                      (2.2)

Шеннон ввел понятие **совершенной секретности** (perfect secrecy): система шифрования обладает совершенной секретностью, если шифртекст не содержит никакой информации об открытом тексте. Формально это означает, что апостериорное распределение открытого текста при известном шифртексте совпадает с априорным распределением:

**P(M = m | C = c) = P(M = m)**                          (2.3)

для всех возможных сообщений m и шифртекстов c. Шеннон доказал, что необходимым условием совершенной секретности является H(K) ≥ H(M) — энтропия ключа должна быть не меньше энтропии сообщения. Единственной практической реализацией является одноразовый блокнот (one-time pad), что делает совершенную секретность практически недостижимой.

В реальных криптосистемах используется понятие **вычислительной стойкости**: алгоритм считается стойким, если не существует полиномиального алгоритма, способного отличить шифртекст от случайной последовательности с вероятностью, существенно превышающей 1/2. Это приводит к следующим информационно-теоретическим критериям качественного шифра:

1. **Равномерность распределения шифртекста**: распределение байтов шифртекста должно быть максимально близко к равномерному, что соответствует максимальной энтропии H(C) → H_max = 8 бит/байт.

2. **Статистическая независимость от открытого текста**: знание открытого текста не должно давать информации о шифртексте — взаимная информация I(M; C) должна стремиться к нулю.

3. **Отсутствие автокорреляции**: последовательные байты шифртекста не должны быть статистически зависимы, что соответствует нулевой автокорреляции.

Эти критерии, сформулированные на языке теории информации, ложатся в основу разрабатываемой методики оценки криптостойкости.
