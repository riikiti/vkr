# 2.3. Статистические тесты криптостойкости

> Статус: ✅ Готово

Энтропийные метрики дополняются набором статистических тестов, оценивающих различные аспекты распределения байтов шифртекста.

## 2.3.1. Частотный анализ (критерий хи-квадрат)

Критерий хи-квадрат (χ²) используется для проверки гипотезы H₀ о равномерности распределения байтов шифртекста. Статистика критерия вычисляется как:

**χ² = ∑ᵢ₌₀²⁵⁵ (Oᵢ − Eᵢ)² / Eᵢ**                     (2.13)

где Oᵢ — наблюдаемая частота байта i, Eᵢ = N / 256 — ожидаемая частота при равномерном распределении, N — общее количество байтов.

При справедливости гипотезы H₀ статистика χ² имеет распределение хи-квадрат с k = 255 степенями свободы. Гипотеза H₀ не отвергается при p-value > 0.05, что свидетельствует о равномерности распределения.

Дополнительно вычисляется **среднее отклонение частоты**:

**δ_freq = (1/256) · ∑ᵢ₌₀²⁵⁵ |Oᵢ/N − 1/256|**          (2.14)

характеризующее среднее абсолютное отклонение эмпирического распределения от идеального равномерного. Для качественного шифра δ_freq → 0.

## 2.3.2. Корреляционный анализ

Корреляционный анализ оценивает наличие линейных и ранговых зависимостей между открытым текстом и шифртекстом, а также последовательных байтов шифртекста.

**Коэффициент корреляции Пирсона** между байтами открытого текста X = {x₁, ..., xₙ} и шифртекста Y = {y₁, ..., yₙ}:

**r = ∑(xᵢ − x̄)(yᵢ − ȳ) / √(∑(xᵢ − x̄)² · ∑(yᵢ − ȳ)²)**  (2.15)

где x̄, ȳ — средние значения. Для идеального шифра r ≈ 0.

**Коэффициент ранговой корреляции Спирмена** вычисляется аналогично, но по рангам значений, что обеспечивает робастность к нелинейным монотонным зависимостям.

**Автокорреляция шифртекста** с лагом lag:

**r(lag) = corr(cᵢ, cᵢ₊ₗₐ₉)**                           (2.16)

Профиль автокорреляции строится для lag = 1, 2, ..., 10. Для идеального шифра r(lag) ≈ 0 при всех значениях лага, что свидетельствует об отсутствии периодических закономерностей.

Для ранжирования вводится балл на основе корреляции:

**S_corr = 1 − |r_Pearson|**                              (2.17)

## 2.3.3. Метрики распределения

Дополнительные статистические характеристики распределения байтов шифртекста:

**Математическое ожидание**:

**μ = (1/N) · ∑ᵢ₌₁ᴺ cᵢ**                                 (2.18)

Для равномерного распределения U(0, 255) идеальное значение μ = 127.5.

**Дисперсия**:

**σ² = (1/N) · ∑ᵢ₌₁ᴺ (cᵢ − μ)²**                        (2.19)

Идеальное значение для U(0, 255): σ² = (256² − 1) / 12 ≈ 5461.25.

**Коэффициент асимметрии (skewness)** характеризует симметрию распределения:

**γ₁ = E[(X − μ)³] / σ³**                                 (2.20)

Идеальное значение γ₁ = 0 (симметричное распределение).

**Эксцесс (kurtosis)** характеризует «тяжесть хвостов» распределения:

**γ₂ = E[(X − μ)⁴] / σ⁴ − 3**                            (2.21)

Для равномерного распределения идеальное значение γ₂ = −1.2.

**Индекс совпадения (Index of Coincidence, IC)** — вероятность того, что два случайно выбранных байта совпадут:

**IC = ∑ᵢ₌₀²⁵⁵ freq(i) · (freq(i) − 1) / (N · (N − 1))**  (2.22)

Для случайных данных IC ≈ 1/256 ≈ 0.00390. Для естественного текста на английском языке IC ≈ 0.065. Шифртекст хорошего алгоритма должен давать IC ≈ 0.00390.
