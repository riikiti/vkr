# 2.4. Тест лавинного эффекта

> Статус: ✅ Готово

Лавинный эффект является одним из ключевых свойств стойкого шифра, обеспечивающим **диффузию** — равномерное распространение влияния каждого бита открытого текста и ключа на все биты шифртекста.

**Строгий лавинный критерий (SAC)**, предложенный Вебстером и Тавалесом (1986), формулируется следующим образом: при инвертировании любого одного бита входных данных каждый бит выходных данных должен изменяться с вероятностью 1/2.

Количественной мерой лавинного эффекта является **лавинный коэффициент**, основанный на расстоянии Хэмминга.

**Расстояние Хэмминга** между двумя двоичными последовательностями одинаковой длины — это количество позиций, в которых соответствующие биты различаются:

**d(c₁, c₂) = ∑ⱼ₌₁ᴸ (c₁ⱼ ⊕ c₂ⱼ)**                     (2.23)

где L — длина последовательности в битах, ⊕ — операция XOR.

**Лавинный коэффициент** определяется как отношение расстояния Хэмминга к общему числу бит:

**AC = d(c₁, c₂) / L**                                    (2.24)

Методика тестирования включает следующие шаги:

1. Зашифровать исходное сообщение m ключом k: c₁ = E(m, k).
2. Инвертировать случайный бит в сообщении m, получив m'.
3. Зашифровать модифицированное сообщение тем же ключом: c₂ = E(m', k).
4. Вычислить лавинный коэффициент AC = d(c₁, c₂) / L.
5. Повторить шаги 1–4 N раз (N = 100) с инвертированием различных бит.
6. Рассчитать статистические характеристики: среднее значение AC_mean, стандартное отклонение AC_std, минимальное AC_min и максимальное AC_max значения.

Интерпретация результатов:
- AC ≈ 0.5 — идеальный лавинный эффект, изменение 1 бита приводит к изменению ~50% бит шифртекста;
- AC < 0.4 — слабый лавинный эффект, алгоритм потенциально уязвим к дифференциальному криптоанализу;
- 0.4 ≤ AC ≤ 0.6 — приемлемый диапазон;
- AC ≈ 0 — отсутствие диффузии, критическая уязвимость.

Для ранжирования вводится балл:

**S_avalanche = 1 − |AC_mean − 0.5| · 2**                 (2.25)

Значение S_avalanche = 1 соответствует идеальному лавинному эффекту (AC = 0.5), а S_avalanche = 0 — полному отсутствию диффузии или 100% изменению бит.

Дополнительно проводится анализ зависимости лавинного коэффициента от размера входных данных, что позволяет оценить стабильность диффузионных свойств алгоритма при различных объемах шифруемой информации.
