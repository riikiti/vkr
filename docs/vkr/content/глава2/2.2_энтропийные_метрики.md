# 2.2. Энтропийные метрики для оценки криптостойкости

> Статус: ✅ Готово

В рамках разрабатываемой методики используются четыре взаимодополняющие энтропийные метрики, каждая из которых оценивает определенный аспект качества шифрования.

## 2.2.1. Энтропия Шеннона

Энтропия Шеннона является основной метрикой оценки случайности шифртекста. Для последовательности байтов C = {c₁, c₂, ..., cₙ} энтропия вычисляется как:

**H(C) = −∑ᵢ₌₀²⁵⁵ p(i) · log₂(p(i))**                (2.4)

где p(i) = freq(i) / N — относительная частота байта i в последовательности, freq(i) — количество вхождений байта i, N — общая длина последовательности.

Для оценки качества шифрования вводится нормализованный **энтропийный балл**:

**S_entropy = H(C) / H_max = H(C) / 8**                 (2.5)

принимающий значения в диапазоне [0, 1]. Значение S_entropy = 1 соответствует идеальному равномерному распределению.

Интерпретация результатов:
- H(C) ≥ 7.9 бит/байт (S_entropy ≥ 0.9875) — хороший шифр, распределение близко к равномерному;
- H(C) < 7.0 бит/байт — слабый шифр, в шифртексте присутствуют статистические закономерности;
- H(C) = 8.0 — идеальный случай, теоретически недостижимый для конечных последовательностей.

Дополнительно проводится **анализ энтропии по блокам**: входная последовательность разбивается на блоки фиксированного размера (по умолчанию 256 байт), и для каждого блока вычисляется энтропия. Равномерность значений энтропии по блокам свидетельствует об однородности шифрования по всей длине данных.

## 2.2.2. Расхождение Кульбака-Лейблера (KL-дивергенция)

KL-дивергенция количественно оценивает отличие эмпирического распределения байтов шифртекста от идеального равномерного распределения. Для распределения P шифртекста и равномерного распределения Q = U(0, 255) KL-дивергенция определяется как:

**D_KL(P ‖ Q) = ∑ᵢ₌₀²⁵⁵ p(i) · log₂(p(i) / q(i))**  (2.6)

где q(i) = 1/256 для всех i.

Для предотвращения неопределенности при p(i) = 0 применяется **сглаживание Лапласа**:

**p̃(i) = (freq(i) + 1) / (N + 256)**                   (2.7)

KL-дивергенция обладает следующими свойствами:
- D_KL ≥ 0 (неотрицательность, неравенство Гиббса);
- D_KL = 0 тогда и только тогда, когда P = Q (распределения совпадают);
- D_KL не является метрикой в математическом смысле (не симметрична и не удовлетворяет неравенству треугольника).

Связь с энтропией Шеннона: при Q = U(0, 255) выполняется соотношение:

**D_KL(P ‖ U) = log₂(256) − H(P) = 8 − H(P)**          (2.8)

Интерпретация:
- D_KL ≈ 0 (< 0.01) — шифртекст практически неотличим от случайной последовательности;
- D_KL > 0.1 — значимые отклонения от равномерности, потенциальная уязвимость.

Для ранжирования вводится балл:

**S_KL = 1 − D_KL / max(D_KL)**                          (2.9)

где max(D_KL) — максимальное значение KL-дивергенции среди всех исследуемых алгоритмов.

## 2.2.3. Условная энтропия

Условная энтропия H(Y|X) оценивает остаточную неопределенность шифртекста Y при известном открытом тексте X. Для вычисления совместного распределения p(x, y) непрерывные значения байтов квантуются в n бинов (по умолчанию n = 16):

**H(Y|X) = −∑ₓ ∑ᵧ p(x, y) · log₂(p(y|x))**            (2.10)

где p(y|x) = p(x, y) / p(x) — условная вероятность, p(x, y) — совместная вероятность попадания значений открытого текста и шифртекста в соответствующие бины.

Интерпретация:
- H(Y|X) ≈ H(Y) — знание открытого текста практически не снижает неопределенность шифртекста (идеальный шифр);
- H(Y|X) ≪ H(Y) — открытый текст содержит значимую информацию о шифртексте (уязвимость).

Отношение H(Y|X) / H(Y) характеризует долю неопределенности, сохраняемую после раскрытия открытого текста. Для качественного шифра это отношение должно быть близко к 1.

## 2.2.4. Взаимная информация

Взаимная информация I(X; Y) количественно оценивает статистическую зависимость между открытым текстом X и шифртекстом Y:

**I(X; Y) = H(Y) − H(Y|X) = H(X) + H(Y) − H(X, Y)**    (2.11)

Взаимная информация обладает свойствами:
- I(X; Y) ≥ 0;
- I(X; Y) = 0 тогда и только тогда, когда X и Y статистически независимы;
- I(X; Y) = I(Y; X) (симметричность);
- I(X; Y) ≤ min(H(X), H(Y)).

Для удобства сравнения вводится **нормализованная взаимная информация**:

**NMI = I(X; Y) / H(Y) ∈ [0, 1]**                        (2.12)

Интерпретация:
- I(X; Y) ≈ 0 (NMI ≈ 0) — полная статистическая независимость, идеальный шифр;
- I(X; Y) > 0 — наличие утечки информации из открытого текста в шифртекст.

Взаимная информация является наиболее строгой метрикой зависимости, так как учитывает не только линейные, но и нелинейные статистические связи между открытым текстом и шифртекстом.
