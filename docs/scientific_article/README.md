# Генерация научных статей

## Общие сведения

Две научные статьи по теме магистерской диссертации «Исследование криптостойкости алгоритмов шифрования с применением энтропийного анализа».

Автор: Курский Р.И.
Научный руководитель (соавтор): Дергачев К.В.
Организация: БГТУ, кафедра ИВТ


## Генерация документов

```bash
# Статья 1
cd docs/scientific_article/article_1
node generate.js

# Статья 2
cd docs/scientific_article/article_2
node generate.js
```

Генераторы используют библиотеку `docx` из `docs/vkr/node_modules/` (путь: `../../vkr/node_modules/docx`).
При каждой генерации предыдущая версия автоматически сохраняется в `versions/` с меткой времени.

## Статья 1: Применение энтропийного анализа для оценки качества криптографических преобразований

- **Тип:** обзорно-теоретическая
- **Формат:** СТНО (по образцу `Rychenkova_STNO_statya.docx`)
- **Связь с диссертацией:** главы 1–2

### Форматирование (СТНО)
- Шрифт: Times New Roman, 12pt
- Поля: 2см со всех сторон
- Межстрочный интервал: одинарный
- Отступ первой строки: 1.25см
- Двуязычная аннотация (RU + EN) и ключевые слова
- УДК: 10pt, по левому краю
- Авторы: 10pt, жирный, по центру
- Подписи таблиц: «Таблица N. Описание», жирный, по правому краю, над таблицей
- Подписи рисунков: «Рис. N. Описание», жирный, по центру, под рисунком
- Библиографический список: 10pt

### Содержание
- Сравнение трёх подходов к оценке криптостойкости (таблица)
- Четыре энтропийные метрики: Шеннон, KL-дивергенция, условная энтропия, взаимная информация
- Система критериев с интегральной формулой оценки
- 1 таблица, 2 рисунка (скриншоты CryptoAnalyzer), 9 источников

## Статья 2: Оценка криптостойкости симметричных шифров методами статистического и энтропийного анализа

- **Тип:** экспериментальная
- **Формат:** AMPU (по образцу `Rychenkova_AMPU.docx`)
- **Связь с диссертацией:** главы 2–4

### Форматирование (AMPU)
- Шрифт: Times New Roman, 14pt
- Поля: левое 3см, правое 1.5см, верхнее/нижнее 2см
- Межстрочный интервал: одинарный
- Отступ первой строки: 1.25см
- Аннотация и ключевые слова только на русском (жирный+курсив)
- Авторы: полные имена, 14pt, жирный
- Заголовки разделов: 14pt, жирный, по центру
- Подписи таблиц: «Таблица N. Описание», жирный, по правому краю, над таблицей
- Подписи рисунков: «Рис. N. Описание», жирный, по центру, под рисунком
- Таблицы: без заливки шапки, рамки 0.5pt, авто-ширина

### Содержание
- 6 алгоритмов: AES-128, DES, 3DES, Blowfish, ChaCha20, ГОСТ 28147-89
- 4 таблицы (характеристики, энтропийный анализ, стат. анализ, итоговый рейтинг)
- 3 рисунка (скриншоты CryptoAnalyzer: энтропия, лавинный эффект, сравнение)
- 9 источников

## Источники литературы

### Критерии подбора
- Давность: не старше 10 лет, предпочтительно 5 лет
- Язык: преимущественно русские авторы
- Тип: рецензируемые журналы, учебные пособия, стандарты (ГОСТ, NIST)
- Формат: ГОСТ Р 7.0.5-2008

- Панасенко С.П. — алгоритмы шифрования (2019)
